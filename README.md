# CODFORM - نماذج الدفع عند الاستلام

تطبيق Shopify لإنشاء نماذج الدفع عند الاستلام بسهولة.

## الميزات

- إنشاء نماذج الدفع عند الاستلام بسهولة
- تخصيص النماذج حسب احتياجاتك
- دعم اللغة العربية
- واجهة مستخدم سهلة الاستخدام
- تكامل مع متجر Shopify الخاص بك

## التثبيت

1. قم بتثبيت التطبيق من متجر Shopify
2. قم بتكوين إعدادات التطبيق
3. ابدأ في إنشاء نماذج الدفع عند الاستلام

## الاستخدام

1. انتقل إلى لوحة التحكم
2. انقر على "إنشاء نموذج جديد"
3. قم بتخصيص النموذج حسب احتياجاتك
4. انشر النموذج على متجرك

## المتطلبات

- متجر Shopify نشط
- خطة Shopify Basic أو أعلى

## الدعم

إذا كنت بحاجة إلى مساعدة، يرجى التواصل معنا عبر البريد الإلكتروني: support@codform.com
