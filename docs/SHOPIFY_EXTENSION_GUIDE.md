
# دليل امتدادات Shopify

هذا الدليل يوثق عملية تطوير ونشر امتدادات Shopify لنماذج الدفع عند الاستلام (COD Form).

## نظرة عامة على امتدادات Shopify

امتدادات Shopify تسمح بإضافة وظائف مخصصة إلى متاجر Shopify. بالنسبة لنا، نستخدم امتدادات السمات (Theme Extensions) لعرض نماذج الدفع عند الاستلام في المتجر.

## طرق إدراج النموذج في متجر Shopify

تتوفر طريقتان رئيسيتان لإدراج نموذج الدفع عند الاستلام في متجر Shopify:

### 1. الإدراج التلقائي

النظام يحاول إدراج النموذج تلقائياً في صفحات المنتجات المحددة. هذه الطريقة مناسبة لمعظم متاجر Shopify التي تستخدم قوالب متوافقة، خاصة قوالب Online Store 2.0.

**ملاحظة مهمة:** قد لا تعمل هذه الطريقة مع جميع القوالب، وخاصة:
- القوالب المخصصة بشكل كبير
- القوالب القديمة التي لا تدعم Online Store 2.0
- القوالب التي تم تعديلها بشكل كبير

### 2. الإدراج اليدوي

يمكن للمستخدم إدراج النموذج يدوياً باستخدام محرر القوالب في Shopify. هذه الطريقة أكثر موثوقية وتعمل مع جميع أنواع القوالب، لكنها تتطلب خطوات إضافية من المستخدم.

**خطوات الإدراج اليدوي:**
1. انتقل إلى لوحة تحكم متجر Shopify
2. اذهب إلى "Online Store" > "Themes" > "Customize"
3. انتقل إلى صفحة المنتج التي تريد إضافة النموذج إليها
4. اضغط على "Add section" واختر "Apps"
5. ابحث عن "نموذج الدفع عند الاستلام" (COD Form) وأضفه
6. أدخل معرف النموذج (Form ID) في إعدادات الكتلة
7. احفظ التغييرات

## هيكل امتداد السمة

مسار الامتداد: `extensions/theme-extension-codform/`

الملفات الرئيسية:
- `shopify.extension.toml`: ملف تكوين الامتداد
- `blocks/codform_form.liquid`: قالب عرض النموذج في المتجر
- `assets/`: مجلد للأصول مثل CSS وJavaScript

### ملف التكوين (shopify.extension.toml)

```toml
name = "CODFORM - نماذج الدفع عند الاستلام"
type = "theme"
handle = "theme-extension-codform"

[[metafields]]
namespace = "codform"
key = "settings"

[settings]
fields = [
  {
    "key": "insertion_method",
    "type": "select",
    "label": { "en": "Form Insertion Method", "ar": "طريقة إدراج النموذج" },
    "options": [
      { "value": "auto", "label": { "en": "Automatic", "ar": "تلقائي" } },
      { "value": "manual", "label": { "en": "Manual", "ar": "يدوي" } }
    ],
    "default": "auto"
  }
]
```

### قالب النموذج (codform_form.liquid)

القالب يتكون من:
1. إعدادات القالب (schema)
2. كود HTML لعرض النموذج
3. نص JavaScript لتحميل وعرض النموذج بشكل ديناميكي

## توافق القوالب

### متوافق تماماً (الإدراج التلقائي يعمل بشكل جيد)
- قوالب Online Store 2.0
- قالب Dawn الرسمي
- معظم قوالب Shopify الحديثة

### قد يتطلب الإدراج اليدوي
- القوالب القديمة (قبل Online Store 2.0)
- القوالب المخصصة بشكل كبير
- القوالب التي تم تعديلها يدوياً بشكل كبير

## استكشاف الأخطاء وإصلاحها

### خطأ 422 عند محاولة الإدراج التلقائي

هذا الخطأ يشير إلى أن القالب لا يدعم طريقة الإدراج التلقائي. الحلول الممكنة:

1. **تبديل إلى الإدراج اليدوي**: اختر "إدراج يدوي" في إعدادات التكامل مع Shopify
2. **التحقق من توافق القالب**: تأكد من أن قالبك يدعم مكونات Online Store 2.0
3. **تحديث القالب**: قم بتحديث قالبك إلى أحدث إصدار يدعم Online Store 2.0

### النموذج لا يظهر في صفحة المنتج

إذا لم يظهر النموذج بعد التكامل الناجح، جرب:

1. **مسح ذاكرة التخزين المؤقت للمتصفح**
2. **التحقق من الإدراج اليدوي**: قم باستخدام الإدراج اليدوي للتأكد من أن النموذج يظهر
3. **التحقق من مصادر الصفحة**: افحص مصادر الصفحة للتأكد من وجود النموذج في HTML
4. **تحقق من سجلات وحدة التحكم**: قد تكون هناك أخطاء في وحدة التحكم تشير إلى المشكلة

### مشاكل شائعة أخرى

1. **تعارض JavaScript**: قد يكون هناك تعارض مع نصوص JavaScript أخرى في القالب
2. **مشاكل CSS**: قد يؤثر تنسيق CSS المخصص في القالب على ظهور النموذج
3. **ترتيب التحميل**: قد يتم تحميل النموذج قبل اكتمال تحميل عناصر الصفحة الأخرى

## أفضل الممارسات لتكامل النموذج

1. **استخدم معرفات فريدة للنماذج**: لتجنب التعارضات في النموذج الواحد
2. **حافظ على بساطة التصميم**: كلما كان تصميم النموذج أبسط، كان أكثر توافقاً
3. **اختبر على قوالب متعددة**: للتأكد من التوافق مع مجموعة متنوعة من القوالب
4. **وفر تعليمات واضحة للإدراج اليدوي**: لمساعدة المستخدمين على إدراج النموذج بسهولة

## حلول متقدمة لمشاكل التوافق

إذا استمرت مشاكل التوافق مع قالب معين، يمكن اتباع هذه الخطوات:

1. **تخصيص طريقة الإدراج**: قم بإنشاء طريقة إدراج مخصصة لهذا القالب المحدد
2. **استخدام السكريبت المخصص**: قم بتطوير سكريبت JavaScript مخصص لإدراج النموذج في هيكل HTML المحدد للقالب
3. **تعديل القالب**: في بعض الحالات، قد تحتاج إلى تعديل قالب المتجر نفسه لدعم النموذج

## خاتمة

يوفر هذا النظام مرونة في إدراج نماذج الدفع عند الاستلام في متاجر Shopify، سواء بطريقة تلقائية أو يدوية. من خلال فهم كيفية عمل الامتدادات والتوافق مع القوالب المختلفة، يمكننا تقديم تجربة سلسة للمستخدمين النهائيين.
